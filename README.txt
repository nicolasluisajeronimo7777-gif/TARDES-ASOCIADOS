Proyecto: TARDES&ASOCIADOS
Estructura de carpetas:
- includes/: Navbar y footer reutilizables.
- assets/css/: estilos personalizados.
- assets/js/: scripts simples de animación.
- *.php: páginas principales (guardarlas en la raíz del sitio).

Cómo usar:
1. Copia la carpeta completa a tu servidor local (ej. C:/xampp/www.tardes&asociados.com).
2. Abre en el navegador: http://localhost/www.tardes&asociados.com/bienvenida.php
3. Las páginas muestran validación de formularios y no dependen de una BD .

Si quieres, puedo añadir un backend mínimo con SQLite para que los registros realmente se persistan localmente.

Se agregó una visualización básica con Chart.js en la página principal para mostrar llegadas tarde (mock data).
